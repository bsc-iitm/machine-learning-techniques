Hilbert's Min-Max Theorem

  • Fact-1: If C is a real symmetric matrix, then it has an orthonormal basis of eigenvectors with real eigenvalues.
  • Fact-2: If C is a symmetric, positive semi-definite matrix, the eigenvalues are non-negative.
  • Fact-3: The covariance matrix is symmetric and positive semi-definite.

From the these three facts, we can conclude that C has a basis of eigenvectors {w1,⋯,wd} with eigenvalues λ1≥⋯≥λd≥0. Now, we are interested in the following optimization problem:

maxw,||w||=1wTCw

Consider any arbitrary vector u that can be uniquely expressed in terms of the orthonormal basis:

u=∑i=1dαiwi

Since we are only looking for vectors with unit norm, ||u||=1. This implies:

uTu=1⟹∑i=1nαi2=1

The above result is true because of the orthonormality of the basis vectors. Next, we can try and evaluate the objective function with u:

uTCu=(∑i=1dαiwi)TC(∑j=1dαjwj)=∑i=1d∑j=1dαiαjwiTCwj=∑i=1dαi2λi≤λ1⋅(∑i=1dαi2)=λ1

We see that the objective function uTCu is bounded above by λ1. This value of λ1 occurs when u=w1. This is one way of proving the theorem. In the steps given above, we have used the fact that Cwi=λiwi, the orthonormality of the eigenvectors and λ1 being the largest eigenvalue.
