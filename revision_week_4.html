Conjugate prior


  • Beta distribution is a conjugate prior for the Bernoulli distribution
  • \(n_1\) ones and \(n_0\) zeros
  • Prior is \(\text{Beta}(\alpha, \beta)\)
  • Posterior is \(\text{Beta}(\alpha + n_1, \beta + n_0)\)

GMM: setup



Two steps

  • \(K\) mixutres: \([\pi_1, \cdots, \pi_k, \cdots, \pi_K]\)
  • Mixture means: \([\mu_1, \cdots, \mu_k, \cdots, \mu_K]\)
  • Mixture variances: \([\sigma_1^2, \cdots, \sigma_k^2, \cdots, \sigma_K^2]\)



Question



What is the value of the following expression? \[ \sum \limits_{i = 1}^{K} \pi_k \]



GMM: generative process

For generating a data-point \(x_i\):

  • Sample a mixture index based on the probabilities \([\pi_1, \cdots, \pi_K]\)
  • Call this index \(k\)
  • \(z_i = k\)
  • Sample a point from the \(k^{th}\) Gaussian
  • \(x_i \sim \mathcal{N}(\mu_k, \sigma_k^2)\)



GMM: density



\[ \Large f_{\tiny GMM}(x) = \sum \limits_{k = 1}^{K} P(z = k) \cdot f(x\ |\ z = k) \]



GMM: density



\[ \Large f_{\tiny GMM}(x) = \sum \limits_{k = 1}^{K} \pi_k \cdot f(x\ |\ z = k) \]



MLE for a GMM



\[ \begin{aligned} L(D; \theta) &= \prod \limits_{i = 1}^{n} f_{\tiny GMM} (x_i)\\\\ l(D; \theta) &= \sum \limits_{i = 1}^{n} \log f_{\tiny GMM}(x_i) \end{aligned} \]



MLE for a GMM

\[ \begin{aligned} l(D; \theta) &= \sum \limits_{i = 1}^{n} \log f_{\tiny GMM}(x_i)\\\\ &= \sum \limits_{i = 1}^{n} \log \left[ \sum \limits_{k = 1}^{K} \pi_k f(x_i\ |\ z_i = k) \right] \end{aligned} \]



MLE for a GMM

\[ \begin{aligned} l(D; \theta) &= \sum \limits_{i = 1}^{n} \log f_{\tiny GMM}(x_i)\\\\ &= \sum \limits_{i = 1}^{n} \log \left[ \sum \limits_{k = 1}^{K} \pi_k f(x_i\ |\ z_i = k) \right]\\\\ &= \sum \limits_{i = 1}^{n} \log \left[ \sum \limits_{k = 1}^{K} \lambda_k^{i} \cfrac{\pi_k}{\lambda_k^{i}} f(x_i\ |\ z_i = k) \right] \end{aligned} \]



MLE for a GMM

\[ \begin{aligned} l(D; \theta) &= \sum \limits_{i = 1}^{n} \log f_{\tiny GMM}(x_i)\\\\ &= \sum \limits_{i = 1}^{n} \log \left[ \sum \limits_{k = 1}^{K} \pi_k f(x_i\ |\ z_i = k) \right]\\\\ &= \sum \limits_{i = 1}^{n} \log \left[ \sum \limits_{k = 1}^{K} \lambda_k^{i} \cfrac{\pi_k}{\lambda_k^{i}} f(x_i\ |\ z_i = k) \right]\\\\\ &\geq \sum \limits_{i = 1}^{n} \sum \limits_{k = 1}^{K} \lambda_k^{i} \log \left[ \cfrac{\pi_k}{\lambda_k^{i}} f(x_i\ |\ z_i = k) \right]\quad \quad \text{Jensen's inequality} \end{aligned} \]



Modified log-likelihood



\[ \Large ml(D; \theta, \lambda) = \sum \limits_{i = 1}^{n} \sum \limits_{k = 1}^{K} \lambda_k^{i} \log \left[ \cfrac{\pi_k}{\lambda_k^{i}} f(x_i\ |\ z_i = k) \right]\\\ \] where


\[ 0 \leq \lambda_k^{i} \leq 1 \quad \text{and} \quad \sum \limits_{k = 1}^{K} \lambda_k^{i} = 1 \]



EM

Parameter estimation for GMMs

  • Intialize \(\theta^{0}\) using Lloyd’s with \(k = K\)

  • Until convergence

    • E-step
      • use \(\theta^{(t)}\) to estimate \(\lambda^{(t)}\)
    • M-step
      • use \(\lambda^{(t)}\) to estimate \(\theta^{(t+1)}\)



E-step



\[ \begin{aligned} \lambda_k^{i} &= P(z_i = k\ |\ x_i)\\\\ &= \cfrac{P(z_i = k) \cdot f(x_i\ |\ z_i = k)}{\sum \limits_{l = 1}^{K} P(z_i = l) \cdot f(x_i\ |\ z_i = l)} \end{aligned} \]



M-step


\[ \Large \begin{aligned} \mu_k = \cfrac{\sum \limits_{i = 1}^{n} \lambda_k^{i} x_i}{\sum \limits_{i = 1}^{n} \lambda_k^{i}} \end{aligned} \]


\[ \Large \begin{aligned} \sigma_k^{2} = \cfrac{\sum \limits_{i = 1}^{n} \lambda_k^{i} (x_i - \mu_k)^2}{\sum \limits_{i = 1}^{n} \lambda_k^{i}} \end{aligned} \]

\[ \Large \pi_k = \cfrac{1}{n} \cdot \sum \limits_{i = 1}^{n} \lambda_k^{i} \]

Hard clustering
