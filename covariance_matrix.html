Covariance Matrix | 2D example

Consider a dataset in R2 with the following covariance matrix:

C=[v1ccv2]

Let w1 and w2 be the eigenvectors of C with eigenvalues λ1 and λ2 respectively, λ1≥λ2. You can think about an image like this:

The variance of the dataset along any direction w is given by:

1n⋅∑i=1n(xiTw)2=wTCw

Using this formula, we can now look at the following variances:

  • Variance along x1: This is the first diagonal element of the covariance matrix.
x1TCx1=[10][v1ccv2][10]=v1
  • Variance along x2: This is the second diagonal element of the covariance matrix.
x2TCx2=[01][v1ccv2][01]=v2
  • Variance along w1: This is the largest eigenvalue.
w1TCw1=w1T(λ1w1)=λ1
  • Variance along w2: This is the second largest eigenvalue.
w2TCw2=w2T(λ2w2)=λ2

The trace of the covariance matrix is the sum of the diagonal elements. This is also equal to the sum of the eigenvalues:

v1+v2=λ1+λ2
  • v1+v2 is the total variance of the dataset computed using the standard basis {x1,x2}.
  • λ1+λ2 is the total variance of the dataset computed using the eigenbasis {w1,w2}.

Both these values are equal. As pointed out in an earlier note, the total variance is invariant to the choice of basis. Now, if we represent all the points in an eigenbasis, then the covariance matrix in this setup will be:

[λ100λ2]

The covariance matrix in the eigenbasis is diagonal. The diagonal entries represent the variance along the principal components and are the eigenvalues. The off-diagonal entries are zero. Recall that the off diagonal entries measure some kind of correlation between two features. Moving to the eigenbasis de-correlates the features. From a matrix decomposition perspective, we have done an eigen-decomposition of the covariance matrix:

C=W⋅[λ100λ2]⋅WT=[||w1w2||]⋅[λ100λ2]⋅[−w1T−−w2T−]=λ1w1w1T+λ2w2w2T
